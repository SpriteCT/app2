# Vulnerability Management System

Система управления уязвимостями с элементами сервисдеска. Прототип фронтенд-приложения с современным темно-фиолетовым дизайном.

## Возможности

### Страница уязвимостей
- Просмотр реестра уязвимостей с фильтрацией
- Поиск по ID, названию, активу
- Фильтрация по критичности и статусу
- Статистика по уровням критичности
- Детальный просмотр уязвимости
- **Импорт результатов сканирования** из различных сканеров (Nessus, OpenVAS, Qualys, Metasploit, Burp Suite)
- **Добавление уязвимостей вручную** с указанием всех параметров
- Классификация по активу, критичности, дате, статусу, ответственному
- Современный темно-фиолетовый интерфейс

### Страница тикетов
- Просмотр всех тикетов с фильтрацией
- **Создание тикетов на основе уязвимостей** - ручное создание с выбором уязвимостей
- Привязка нескольких уязвимостей к одному тикету
- Фильтрация по приоритету и статусу
- Отслеживание сроков выполнения (с индикацией просроченных)
- **Встроенный чат для каждого тикета** с историей обсуждений
- Статистика по приоритетам и статусам
- **Назначение и изменение ответственных лиц** прямо из модального окна тикета
- Детальная информация о тикете с историей обсуждений

### Страница активов
- Просмотр всех активов с фильтрацией по типу, статусу, критичности
- Поиск по ID, имени или IP-адресу
- Добавление активов вручную с полной классификацией
- Импорт активов из CMDB (ServiceNow, Jira, Azure DevOps, CSV, Excel)
- Классификация по: типу (сервер, рабочая станция и т.д.), критичности, принадлежности
- Отслеживание состояния актива (в эксплуатации, недоступен, выведен из эксплуатации)
- Привязка активов к уязвимостям и тикетам
- Детальная информация об активе со связанными уязвимостями и тикетами
- Экспорт активов для сканеров (.csv)
- Статистика по состояниям активов

### Страница клиентов и проектов
- Просмотр карточек всех клиентов с краткой информацией
- Детальная информация о клиенте в модальном окне
- **Контактная информация**: основной контакт с должностью, дополнительные контакты, телефоны и email
- **Договор и SLA**: номер договора, уровень SLA, даты подписания и окончания, биллинг цикл
- **Инфраструктура**: количество серверов, рабочих станций, сетевого оборудования, тип развертывания (Cloud/On-Premise)
- **Проекты клиента**: список всех проектов с типом, статусом, приоритетом, прогрессом
- Возможность ведения нескольких проектов одновременно для одного клиента
- Типы проектов: Vulnerability Scanning, Penetration Test, Network Scanning, BAS, Web Application Scanning, Compliance Check
- Создание новых клиентов и проектов
- Поиск по названию, контактному лицу или отрасли
- Привязка уязвимостей, тикетов и активов к проектам

### Страница отчётов и аналитики
- **Дашборды с визуализацией данных**: графики уязвимостей, тикетов, активов
- **Статистика в реальном времени**: количество уязвимостей, активных тикетов, устранённых уязвимостей
- **KPI специалистов**: эффективность работы, среднее время решения, количество решённых тикетов
- **Динамика уязвимостей**: тренды по месяцам с разбивкой по критичности
- **Формирование отчётов по шаблонам**: еженедельный, ежемесячный, ежеквартальный, отчёт для клиента
- **Автоматическая генерация отчётов** для клиентов
- **Экспорт данных** в различные форматы (PDF, Excel, Word)
- **Интерактивные графики** (Pie Chart, Bar Chart, Line Chart) с использованием Recharts

### Разделение по клиентам
- Переключатель клиентов в навигационной панели
- Выбор "Все клиенты" для просмотра всех данных
- Фильтрация уязвимостей, тикетов и активов по выбранному клиенту
- Статистика адаптируется под выбранного клиента
- Поддержка множественных клиентов интегратора

## Установка и запуск

### Требования
- Node.js 18+
- npm или yarn

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

### Сборка для продакшена
```bash
npm run build
```

Собранные файлы будут в директории `dist/`

## Технологии

- **React 18** - UI библиотека
- **Vite** - Сборщик
- **Tailwind CSS** - Утилитарный CSS
- **Lucide React** - Иконки
- **Recharts** - Библиотека для графиков и визуализации данных

## Реализованная функциональность

✅ Управление активами (импорт из CMDB, классификация, отслеживание состояния, привязка к уязвимостям)  
✅ Реестр уязвимостей (импорт из сканеров, ручное добавление)  
✅ Управление тикетами (автоматическое создание, назначение ответственных, встроенный чат)  
✅ Управление клиентами и проектами (карточки клиентов, контакты, SLA, договоры, инфраструктура, управление проектами)  
✅ **Отчётность и аналитика** (дашборды с графиками, KPI специалистов, формирование отчётов, автогенерация)

## Цветовая схема

Приложение использует темно-фиолетовую цветовую схему:

- **Фон**: `#0a0e27` - темно-синий фон
- **Поверхность**: `#131636` - темная поверхность
- **Карточки**: `#1a1f3f` - карточки
- **Границы**: `#252a50` - границы
- **Purple Primary**: `#6b46c1` - основной фиолетовый
- **Purple Secondary**: `#8b5cf6` - вторичный фиолетовый
- **Purple Accent**: `#a78bfa` - акцент
